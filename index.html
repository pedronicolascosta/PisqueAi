Pisque_Ai, o software para escrever com os olhos.

O que começou como um projeto de pesquisa escolar, o Pisque Aí hoje é uma ferramenta capaz de permitir que pessoas portadoras de Esclerose Lateral Amiotrófica consigam se comunicar com o movimento ocular.
